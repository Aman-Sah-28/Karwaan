# Karwaan – Every Journey Has a Story

## Project Overview

Karwaan is a responsive tourism web application designed to help travelers discover destinations, plan journeys, share travel experiences, and utilize smart travel tools. The project was developed as part of the Full Stack Development (MCA413-1) CIA Assessment.

## Problem Statement

Most tourism websites focus only on displaying destinations and travel information. Travelers often lack interactive tools for trip planning, destination discovery, weather checking, itinerary creation, and experience sharing.

Karwaan addresses these challenges through an interactive and user-friendly tourism platform.

## Objectives

* Provide destination discovery features
* Enable trip planning and budget management
* Offer weather information
* Allow travelers to share experiences
* Demonstrate HTML5 APIs and JavaScript concepts
